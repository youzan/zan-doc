exec()

exec(string  $execfile, array  $args) : boolean

exec 执行一个外部程序，此函数是 exec 系统调用的封装 执行成功后，当前进程的代码段将会被新程序替换。子进程蜕变成另外一套程序。父进程与当前进程仍然是父子进程关系。 父进程与新进程之间可以通过可以通过标准输入输出进行通信，必须启用标准输入输出重定向。

Parameters

string $execfile

指定可执行文件的绝对路径，如 "/usr/bin/python"
array $args

数组，是 exec 的参数列表，如 array('test.py', 123)，相当与 python test.py 123

Returns

boolean —

成功返回 true，失败返回 false
